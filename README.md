# cybergraph

# Set Up

0. Install foundry:
   `curl -L https://foundry.paradigm.xyz | bash`

1. Upgrade your foundry
   `foundryup`

2. To enable husky pre-commit
   `yarn && yarn prepare`

3. To install contract dependencies
   `forge install`

4. To build
   `forge build`

5. To test
   `forge test -vvv`

6. (optional) To run static analysis
   `slither src/`

7. To see contract sizes
   `yarn size`
